10. GitHub Personal Access Token (issue #11)

Should NOT show icon — this is a developer credential being displayed

Regression test for issue #11: the extension was overwriting a freshly-generated GitHub PAT with a 6-digit TOTP code. Two variants below — both should be ignored.

Variant A: readonly display of generated PAT

readonly + aria-label "Personal access token" — should be excluded by readonly skip and PAT exclusion.

Variant B: writable token-name input (defensive)

name contains "access-token" — should be excluded by PAT exclusion pattern.

Variant C: API key field

name="api-key" — should be excluded by API key exclusion pattern.
